## Project Overview

Corporate website for an industrial equipment company specializing in factory logistics solutions, European/American advanced equipment imports, industrial doors, garage doors, and pharmaceutical equipment.

## Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Package Manager**: npm

## Commands

- `npm run dev` - Start development server
- `npm run build` - Production build
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

## Architecture

- `src/app/` - App Router pages and layouts
- `src/components/` - Reusable UI components
- `public/` - Static assets (images, icons)
- `tailwind.config.ts` - Tailwind configuration with custom theme
